# 🌱 AgriSense – Smart Agriculture Advisory Web Application

## Overview
AgriSense is a smart agriculture advisory web application designed to help farmers make informed decisions using modern technology. The platform provides crop recommendations, weather updates, soil information, market prices, and AI-powered pest detection through an easy-to-use interface.

This project was developed to demonstrate how web technologies and AI can be combined to solve real-world agricultural challenges.

---

## Features

- 🌾 Crop recommendations based on user inputs
- 🌦️ Real-time weather updates
- 🌱 Soil moisture and soil health information
- 🐛 AI-powered pest detection
- 💰 Live market price updates for crops
- 📱 Responsive and user-friendly interface
